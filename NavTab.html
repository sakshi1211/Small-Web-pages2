HTML stands for Hyper Text Markup Language, which is used to create the structure of Web Page or Website.

CSS stands for Cascading Style Sheet, which is used for styling the document.

JavaScript is a programming language used for scripting the web pages, making them interactive and dynamic.

Bootstrap is an open source, cross platform library to build interactive UI rapidly.
